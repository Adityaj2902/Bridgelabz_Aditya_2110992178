AJAX and API Calls

XML HTTP Request

In JavaScript, there are multiple ways of doing AJAX calls. We're going to start with the most old school one, which is called the XML HTTP request function.

		var request = new XMLHttpRequest();

Opening the Request

We need to open the request and specify the type of request (GET, POST, etc.) and the URL to which the request should be made.

		request.open('GET', 'https://restcountries.eu/rest/v2/name/portugal');

Sending the Request

We need to send the request using the `send()` method.

		request.send();

Registering a Callback Function

We need to register a callback function to handle the response from the API.

		request.addEventListener('load', function() {
			var data = JSON.parse(this.responseText);
			// Do something with the data
		});

API Endpoint

We're using the Rest Countries API, which provides information about countries.

		https://restcountries.eu/rest/v2/name/portugal

Converting JSON to a JavaScript Object

We need to convert the JSON response to a JavaScript object using `JSON.parse()`.

		var data = JSON.parse(this.responseText);
